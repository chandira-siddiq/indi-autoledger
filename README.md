# Indi AutoLedger

Indi AutoLedger is a Front end application designed to provide User Interface for a application that delivers Registerd Vehicle data. This project uses React and Vite for fast development and modern UI.

## Project Structure
```
indi-autoledger/
├── public/
├── src/
│   ├── App.jsx
│   ├── main.jsx
│   ├── assets/
│   ├── components/
│   ├── services/
│   ├── styles/
├── index.html
├── package.json
├── vite.config.js
├── README.md
```
